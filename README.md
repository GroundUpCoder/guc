# guc
 The GroundUpCoder Python library

A pure Python utility library for making games with Pygame.

# INSTALLATION

```
pip install guc
```

Either `pygame` or `pygame-ce` must also be installed.

# TESTING

To test `guc`, run

```
python -m unittest
```

All tests live in `test_guc`
